# Таймер обратного отсчета

Простой таймер обратного отсчета, реализованный с помощью EWW (ElKowar's Wacky Widgets).

## Функциональность

- Таймер обратного отсчета с возможностью задать произвольное время
- Поддержка различных форматов ввода времени:
  - Число минут (например, `5` - 5 минут)
  - Минуты с буквой (`5m` - 5 минут)
  - Часы и минуты (`1h30m` - 1 час 30 минут)
  - Формат MM:SS (`1:30` - 1 минута 30 секунд)
- Предустановленные таймеры: 5, 10, 30 минут
- Возможность ввода произвольного времени через диалог Yad
- Уведомления по окончании таймера
- Звуковое оповещение по окончании таймера

## Зависимости

- eww (ElKowar's Wacky Widgets)
- yad (Yet Another Dialog)
- notify-send (libnotify)

## Установка

Для установки таймера на другой системе выполните:

```bash
./install_timer.sh
```

Скрипт установки:
- Проверит и установит зависимости (eww, yad, libnotify)
- Скопирует все необходимые файлы в директорию конфигурации
- Настроит права доступа
- Создаст ярлык в меню приложений

## Использование

- Кнопки "5 мин", "10 мин", "30 мин" - установка предустановленного времени
- Кнопка "Произвольно" - ввод произвольного времени через диалог Yad
- Кнопка "▶ Старт" - запуск таймера
- Кнопка "■ Стоп/Сброс" - остановка и сброс таймера
- Кнопка "✕" - закрытие окна таймера

## Добавление на панель XFCE (опционально)

Для добавления таймера на панель XFCE:

1. Щелкните правой кнопкой мыши по панели XFCE
2. Выберите "Параметры панели" → "Добавить новые элементы"
3. Найдите "EWW Timer" или "Launcher" с командой запуска
4. Добавьте на панель

Также можно создать ярлык запуска в меню приложений и перетащить его на панель.

## Файлы

- `countdown_timer.sh` - основной скрипт управления таймером
- `yad_input.sh` - скрипт для ввода времени через Yad
- `run_timer.sh` - скрипт запуска таймера
- `eww.yuck` - описание интерфейса EWW
- `eww.scss` - стили EWW
- `timer_toggle.sh` - скрипт переключения видимости таймера
- `test_yad_functionality.sh` - тест функциональности yad
- `view_logs.sh` - просмотр логов
- `install_timer.sh` - скрипт установки
