# Система таймера с вводом через Yad

## Описание

Система позволяет вводить числа через диалог Yad и использовать их для установки времени таймера.

## Функциональность

- Ввод любого числа через Yad диалог
- Преобразование числа в секунды (число интерпретируется как минуты)
- Поддержка различных форматов:
  - Простое число: `5` → 5 минут
  - Минуты: `10m` → 10 минут
  - Часы: `1h` → 1 час
  - Комбинированный: `1h30m` → 1 час 30 минут
  - Формат времени: `5:30` → 5 минут 30 секунд

## Тестирование

### С помощью тестового скрипта:

1. Запустите тестовый скрипт:
   ```bash
   ./timer/test_yad_input.sh
   ```

2. Введите число в диалоговое окно Yad

3. Проверьте результат в логах:
   ```bash
   ./timer/view_logs.sh
   ```

### Через интерфейс Eww:

1. Запустите интерфейс таймера:
   ```bash
   eww -c /home/ig_ro/Документы/eww/timer open timer
   ```

2. Нажмите кнопку "Произвольно"
3. Введите число в диалог Yad
4. Нажмите "Применить" и затем "Старт"

## Логирование

Все действия системы записываются в файл `/tmp/timer_debug.log` для отладки.
